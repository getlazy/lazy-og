# Lazy Engine Generator
Generator for lazy engines. 

`generator-lazy-engine` creates a base template for a new, empty engine that can be deployd to [Lazy](http://getlazy.org/).


## Install

Install `yeoman`, if you don't have it:
```
npm install -g yo
```

Install `generator-lazy-engine` 

``` 
$ npm install --global generator-lazy-engine
```


## Usage

```
$ yo lazy-engine
```

*Note that this template will generate files in the current directory, so be sure to change to a new directory first if you don't want to overwrite existing files.*


### Running tests

Once the project is scaffolded, inside the project folder run:

```
$ npm -g install mocha
$ mocha
```
